# 📄 Selection Sort

## 🔍 Como funciona
O **Selection Sort** encontra o menor elemento e o coloca na posição correta, repetindo para o restante da lista.

## 📦 Complexidade de Espaço
- **O(1)** – Algoritmo in-place.

## 📈 Análise Assintótica
- **Todos os casos:** O(n²)

## 🔁 Exemplo de Iteração
Lista: `[5, 3, 8, 4, 2]`
- Passo 1: `[2, 3, 8, 4, 5]`
- Passo 2: `[2, 3, 8, 4, 5]`
- Passo 3: `[2, 3, 4, 8, 5]`
- Passo 4: `[2, 3, 4, 5, 8]`

## 📊 Conjunto de Dados Ideal
- Listas pequenas.
