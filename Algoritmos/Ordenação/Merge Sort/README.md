# 📄 Merge Sort

## 🔍 Como funciona
Divide a lista ao meio, ordena recursivamente e mescla.

## 📦 Complexidade de Espaço
- **O(n)** – Precisa de espaço extra para as sublistas.

## 📈 Análise Assintótica
- **Todos os casos:** O(n log n)

## 🔁 Exemplo de Iteração
Lista: `[5, 3, 8, 4, 2]`
- Divide: `[5, 3]`, `[8, 4, 2]`
- Ordena: `[3, 5]`, `[2, 4, 8]`
- Mescla: `[2, 3, 4, 5, 8]`

## 📊 Conjunto de Dados Ideal
- Listas grandes.
- Dados em arquivos (externo).
